Installer

Community-made App Package Installer for KaiOS

How to use?

Open this app with a valid installation package (OpenWebApp, OmniSD, AIB or .zip) from an enabled file manager app to sideload apps.

The app first attempts to use device APIs, before attemtping self-debug, and ultimately data patching to install packages.
